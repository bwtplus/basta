# Basta Web

## Basta web on local

### CLI webpack
    //first run install
	> npm install
    //webpack build - only static assets
	> npm run build
	> npm run prod
	> npm run watch
### CLI hugo
    //start dev server
	> hugo server
    //simple build site
	> hugo
